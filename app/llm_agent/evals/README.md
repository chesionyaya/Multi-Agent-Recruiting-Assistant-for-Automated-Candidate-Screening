# Evaluation Notes

Recommended evaluation sets:

1. Structured CV + clear JD (rule-friendly).
2. Unstructured CV paragraphs (LLM-friendly).
3. Mixed language resumes.
4. Edge cases with missing sections.

Track these metrics:

- skill precision/recall vs labeled truth
- decision agreement vs recruiter label
- explanation usefulness (human rating)
- route stability (`rule/llm/hybrid`) and confidence
